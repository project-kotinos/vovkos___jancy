Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This struct holds information about single address of the Pcap
device.

A list of network adapter addresses is part of Pcap device description
stored in `io.PcapDeviceDesc`.



.. rubric:: See also:

`io.PcapDeviceDesc`, `io.createPcapDeviceDescList`
