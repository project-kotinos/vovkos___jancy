bitflag enum io.NamedPipeEvents
===============================

Overview
~~~~~~~~

This enumeration contains codes of all possible events that might
happen on `io.NamedPipe` object. The set of *active* events is accessible
via `io.NamedPipe.m_activeEvents`. :ref:`More...<details-type_io_namedpipeevents>`
