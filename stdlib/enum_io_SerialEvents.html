Detailed Documentation

This enumeration contains codes of all possible events that might happen on io.Serial object. The set of active events is accessible via io.Serial.m_activeEvents.

See also:

io.Serial

Enum Values

IoError

An IO error occured on the serial port; detailed information about the error is available at io.Serial.m_ioError.

IncomingData

Incoming data is available on the serial port; this data could be fetched with io.Serial.read method.

ReadBufferFull

Serial port receive buffer is full; upcoming data may be dropped .
