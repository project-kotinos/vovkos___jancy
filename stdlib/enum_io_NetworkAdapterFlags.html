Detailed Documentation

This enumeration specifies flag properties of the network adapter.

When enumerating available network adapter you first call io.createNetworkAdapterDescList, then iterate through the resulting list of io.NetworkAdapterDesc structures, each contains a set of flag properties in io.NetworkAdapterDesc.m_flags field.

See also:

io.createNetworkAdapterDescList, io.NetworkAdapterDesc

Enum Values

Dhcp

DHCP (Dynamic Host Configuration Protocol) is enabled on this adapter.

Ddns

DDNS (Dynamic DNS) is enabled on this adapter.

This adapter supports broadcast.

Multicast

This adapter supports multicast.

Ip4
