bitflag enum io.FileStreamEvents

Overview

This enumeration contains codes of all possible events that might happen on io.FileStream object. The set of active events is accessible via io.FileStream.m_activeEvents. Moreâ€¦

bitflag enum FileStreamEvents
{
    IoError,
    IncomingData,
    ReadBufferFull,
    WriteBufferEmpty,
    WriteBufferReady = WriteBufferEmpty,
    Eof,
};

Detailed Documentation

This enumeration contains codes of all possible events that might happen on io.FileStream object. The set of active events is accessible via io.FileStream.m_activeEvents.

See also:

io.FileStream

Enum Values

IoError

An IO error occured on the file stream; detailed information about the error is available at io.FileStream.m_ioError.

IncomingData

More data is available on the file stream; this data could be fetched with io.FileStream.read method.

ReadBufferFull

File stream receive buffer is full; upcoming data may be dropped (depending on the underlying device represented by the file).

WriteBufferEmpty

File stream transmit buffer is empty.

WriteBufferReady

File stream transmit buffer is ready to accept more data; itâ€™s OK to issue more io.FileStream.write requests.

Eof

End-of-file was reached on the file stream.
