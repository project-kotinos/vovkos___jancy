bitflag enum io.UsbEndpointEvents

Overview

bitflag enum UsbEndpointEvents
{
    IoError,
    IncomingData,
    ReadBufferFull,
    WriteBufferEmpty,
    WriteBufferReady = WriteBufferEmpty,
    WriteCompleted,
};

Detailed Documentation

Enum Values

IoError

An IO error occured on the USB endpoint; detailed information about the error is available at io.UsbEndpoint.m_ioError.

IncomingData

More data is available on the USB endoint; this data could be fetched with io.UsbEndpoint.read method.

ReadBufferFull

IN-endpoint receive buffer is full; upcoming data may be dropped (depending on the endpoint transfer type).

WriteBufferEmpty

OUT-endpoint transmit buffer is empty.

WriteBufferReady

OUT-endpoint transmit buffer is ready to accept more data; itâ€™s OK to issue more io.UsbEndpoint.write requests.

WriteCompleted

OUT-endpoint has no active or pending USB transfers. Useful when you need to wait for all outbound transfers to complete before closing an endpoint.
